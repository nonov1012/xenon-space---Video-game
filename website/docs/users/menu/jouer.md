# Menu de Personnalisation --- Guide utilisateur

## À quoi sert ce menu ?

Le **menu de personnalisation** est l'écran principal où vous configurez les règles et les unités de votre prochaine partie de **Xenon Space** avant de la lancer.

Il permet de :
* Ajuster les **paramètres de la carte** (planètes, astéroïdes, argent de départ).
* Personnaliser en détail les **statistiques de vos vaisseaux**.
* **Lancer la partie** avec les configurations choisies.
* **Réinitialiser** les paramètres.
* **Retourner au menu principal**.

---

## Ce que vous voyez à l'écran

Le menu est divisé en deux sections principales : le **Panneau Central de Configuration** et les **Boutons d'Action** (en bas).

### Panneau Central de Configuration (Onglets)

| Onglet | Contenu |
| :--- | :--- |
| **Classique** | Contient les paramètres de base pour la carte (ex: Nombre de planètes, Nombre d'astéroïdes). |
| **Avancé** | Contient tous les paramètres de la carte, y compris les niveaux de base de départ et l'argent de départ. |
| **Vaisseaux** | Permet d'éditer les statistiques des différentes classes de vaisseaux (PV, Attaque, Vitesse, etc.). |

### Boutons d'Action (en bas)

* **JOUER** : Lance la partie immédiatement avec les paramètres actuels.
* **RESET** : Rétablit tous les paramètres de la carte à leurs valeurs par défaut.
* **RETOUR MENU** : Quitte l'écran de personnalisation pour revenir au menu principal.

---

## Comment Naviguer et Modifier les Paramètres

### A. Navigation

| Action | Résultat |
| :--- | :--- |
| **Clic sur un onglet** | Change la vue pour afficher les options spécifiques à cet onglet. |
| **Molette de la souris** | Permet de faire défiler la zone de paramètres vers le haut ou le bas (utile dans l'onglet **Vaisseaux**). |
| **Survol d'un bouton** | Le bouton s'anime (grossissement et son de survol). |

### B. Modifier les Paramètres de Partie (Onglets Classique / Avance)

1.  **Ajuster un paramètre :** Cliquez et maintenez le **curseur rond** sur le *slider* (la barre de défilement) et déplacez-le horizontalement pour choisir une valeur entre **Min** et **Max**.
2.  **Bouton RANDOM :** Cliquez sur le bouton **RANDOM** pour activer ou désactiver l'attribution de valeurs aléatoires pour tous les paramètres de l'onglet actif (dans les limites Min/Max).

### C. Modifier les Statistiques de Vaisseaux (Onglet Vaisseaux)

1.  **Sélection du Vaisseau :**
    * Cliquez sur le **Dropdown** (la boîte en haut à gauche) pour ouvrir la liste des vaisseaux.
    * Cliquez sur le nom du vaisseau souhaité (ex : **Fighter**, **MotherShip**, **Foreuse**).
    * *Pour le MotherShip uniquement :* Vous devez ensuite choisir le **Tier (1, 2, 3, 4)** pour modifier les statistiques de ce niveau d'amélioration.
2.  **Ajustement des Statistiques :**
    * **Sliders :** Pour la plupart des valeurs (PV Max, Vitesse, etc.), utilisez le curseur pour ajuster la valeur.
    * **Boutons +/- :** Pour les paramètres spécifiques (ex: Portée, Capacité de Vaisseaux), utilisez les boutons **[ - ]** et **[ + ]** pour incrémenter ou décrémenter la valeur par pas.
    * Si la liste est longue, utilisez la **molette de la souris** pour faire défiler la zone des statistiques.

---

## Conseils d'utilisation

* **Vérifiez le Tier :** Si vous personnalisez le **MotherShip**, toute modification n'affecte que le **Tier** actuellement sélectionné.
* **Le RESET :** Le bouton **RESET** ne réinitialise que les paramètres de la carte (onglets Classique/Avancé), pas les statistiques des vaisseaux.
* **Lancement :** Lorsque vous cliquez sur **JOUER**, toutes les valeurs affichées sont immédiatement utilisées pour le lancement de votre nouvelle partie.

## En résumé

| Situation | Action |
| :--- | :--- |
| Changer le niveau d'argent de départ. | Onglet **Avancé** → Ajustez le slider *Argent depart*. |
| Augmenter l'Attaque de votre **Fighter**. | Onglet **Vaisseaux** → Sélectionnez *Fighter* → Utilisez le slider ou les boutons pour ajuster l'*Attaque*. |
| Lancer la partie. | Cliquez sur **JOUER** en bas de l'écran. |
| Revenir au menu principal. | Cliquez sur **RETOUR MENU** en bas de l'écran. |
