## ⚠️ Important !

> 🔧 Le jeu est en cours de développement et donc contenir des bugs.

## 📖 Présentation

**Xenon Space** est un jeu de type combat spatial en 2D tour par tour développé en Python avec Pygame.

🎮 Combattez des vagues d'ennemis dans l'espace et survivez le plus longtemps possible !

## 📦 Installation

Le jeu utilise les bibliothèques suivantes :

- ![Pygame](https://img.shields.io/badge/pygame-00979D?style=flat-square&logo=python&logoColor=white) **pygame**
- ![pypresence](https://img.shields.io/badge/pypresence-5865F2?style=flat-square&logo=discord&logoColor=white) **pypresence**

Pour les installer, exécutez la commande suivante :
```bash
pip install -r requirements.txt
```

## 🎮 Lancer le jeu

```bash
python main.py
```
